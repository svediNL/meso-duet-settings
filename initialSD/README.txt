Duet V1.03 SD-image notes.

1. The configuration files on the uSDcard as supplied with new tested and sealed Duet V1.03 controllers is strictly for factory test purposes, and for connecting your Duet Wifi to your network. You MUST generate your own configuration files - if new to this we suggest you use the configuration tool at https://configurator.reprapfirmware.org/
2. Duet V1.03 boards released from December 2017 onwards are flashed with the latest stable firmware release, V1.19.2, downloadable from https://github.com/dc42/RepRapFirmware/releases.
3. To connect to a Wifi network see https://www.duet3d.com/wiki/Getting_connected_to_the_Duet_WiFi#Connecting_to_your_access_point_using_firmware_1.19_or_later
